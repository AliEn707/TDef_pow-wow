Tdef::Log - Логи, содержат инфу о произведенных действиях
{
  action - действие: появление/выход обьекта в системе, подключение к серверу игрока
  player_id - игрок с которым связано действие
  object_id - обьект действие
  object_type - тип обьекта
  value - количество
  other - другое
}
index: action, player_id, object_id object_type, ?action player_id, ?action object_id object_type

action:
  income - вход обьекта в систему (создание обьекта)
  outcome - выход обьекта из сестемы (разрушение обьекта)
?  take - получение обьекта от игрока
?  give - отдача обьекта игроку
  server - вход игрока на сервер
  login - логин игрока
