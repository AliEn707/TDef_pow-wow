Tdef::Resource - один ресурс
{
  name - название
  description -описание
  type - тип обьекта
  cost - цена для покупки
  sell - цена при продаже
}

Tdef::Resource::User - ресурсы принадлижащие пользователю 
{
  user_id - владелец
  resource_id - ресурс
  quantity - количество
  capacity - максимально доступное количество
}

абсолютно максимальное количество ресурсов 2000000000
