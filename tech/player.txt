Tdef::Player::Auth - модель хранящая токен для авторизации пользователя
{
  player_id integer
  token integer
}
index: player_id

Tdef::Player::Item- обьекты принадлижащие пользователю 
{
  player_id - владелец
  item_id - ид обьекта
  item_type - тип обьекта
  quantity - количество
  capacity - максимально доступное количество
}
index: player_id, item_id item_type, quantity

seeds: при добавлении новых обектов для каждого игрока должны быть созданы items по каждому из типов, с заданой вместимостью и количеством 0.
абсолютно максимальное количество обьектов одного типа 2000000000
